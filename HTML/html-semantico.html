Elementos semanticos

Dentro del mundo del HTML hay ciertas etiquetas cuya presencia no presenta ningun cambio fisico o visual en la estructura de la pagina web, a simple vista por lo menos.

Dichos elementos nos permiten indicarle al navegador que tipo de elementos queremos mostrar y en que forma queremos que sean representados.

"article"

Indica un articulo de la pagina

"aside"

Es el tipico menu dentro de las paginas web que se presenta en una barra lateral indicando diferentes secciones de la web

"figcaption"

Indica que dentro de dicho elemento viene informacion acerca de la imagen que luego va a ser introducida dentro del siguiente elemento

"figure"

El elemento que contiene la etiqueta "img" la cual permite añadir imagenes dentro de la web

"footer"

Indica que todo lo contenido dentro de esta etiqueta será el pie de pagina de la pagina

"header"

Al igual que el footer, pero con el encabezado, indica que dentro de el iran los elementos contenidos en el encabezado de nuestra pagina

"main"

Indica cual es realmente el contenido principal de la pagina web, util para que motores de busqueda como Google puedan reconocer si nuestra pagina es apropiada para tal resultado de busqueda

"nav"

Indica al navegador que dentro de dicha etiqueta se encuentra un menu de navegación, etiqueta altamente utilizada en conjunto con CSS

"section"

Marca diferentes secciones de una pagina

"time"

Como lo indica su nombre, indica que dentro de este elemento habrá algo relacionado con el tiempo.

"div"

En si, no es un elemento semantico pero si contenedor, pero en lo visual y a efectos practicos no son muy distintos, no preformatea nada en la pagina, no genera margenes, ni nada.

Pero nos ayuda a organizar y dividir diferentes secciones, generalmente similares de la pagina para poder darle caracteristicas diferentes, como margenes, colores de fondo, etc.
