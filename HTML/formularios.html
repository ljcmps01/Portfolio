Volver al inicio

Hasta el momento todo lo visto sirve unicamente para mostrarle información al usuario, sean imagenes, texto etc.

Sin embargo en HTML existen los formularios los cual nos permiten obtener información del usuario, todo esto, utilizando la etiqueta "form"

Elemento form

parametros

action

Presentado en la forma de action="[URL]" , se utiliza para indicar hacia donde será dirigida la información obtenida del usuario suele ir a paginas PHP o JS (javascript)

method

Presentado en la forma de method="[GET/POST/PUT/DELETE]". Nos permite indicar que hacer con la información recibida

Elementos del formulario

label

Lo utilizamos para asignar variables donde se contendran los valores ingresados por el usuario.

Se presenta en la forma <label for="[nombre de variable]"> [texto]</label>

Se usa usualmente junto al siguiente elemento, input

input

Ahora si, con este elemento le permitiremos al usuario ingresar información util para el funcionamiento de nuestra pagina web

Se presenta en la forma <input type="[text]">

Esto nos crea una caja de texto con caracteristicas dependientes del parametro type ingresado.

ejemplo:

<label for="username"> Nombre de usuario:</label>
<input type="[text]">

Da como resultado:

Obviamente, no sirve unicamente para ingresar texto, esto se puede modificar segun nuestras necesidades modificando el valor del parametro type:

  • text: como vimos, se utiliza para ingresar caracteres alfanuméricos
  • number: se utiliza para ingresar UNICAMENTE números (enteros)
  • password: se utiliza para ingresar contraseñas alfanuméricas pero sin poder visualizar lo que ingresamos.
  • email: a niveles visuales es lo mismo que el tipo texto, sin embargo con otro elemento de la etiqueta input podemos verificar si el texto ingresado es efectivamente un email (a nivel sintactico)

select

Este elemento nos permite crear una lista desplegable con varias opciones encadenadas para que el usuario pueda elegir UNA de ellas.

Dichas opciones van dentro del elemento option

Se presenta de la forma

< select name=" " id=" ">
< option value="[text]">[text]< /option>
</select>

textarea

Es un elemento similar al text sin embargo este elemento posee argumentos que nos permiten modificar el tamaño del cuadro de texto, ideal para cuando necesitamos que el usuario introduzca grandes cantidades de texto, por ejemplo, una caja de comentarios

Se presenta de la forma:

< textarea name="[name]" id=" " cols="[# de columnas]" rows="[# de filas]" >< /textarea>

input type="file" y parametro enctype

Hay un 5to valor posible del input type anteriormente mencionado, file, este nos permite subir archivos a la pagina web de la siguiente forma:

Es importante aclarar que este tipo de archivo debe ir acompañado de un parametro dentro del elemento form, el enctype, que puede tomar 3 valores posibles:

  • application/x-www-form-urlenconded: sirve para cargar scripts que puedan ser procesados por el navegador (archivos .js; .php;etc)
  • multipart/form-data: Permite subir cualquier tipo de archivos
  • text/plain: permite subir unicamente archivos de texto plano (.txt)
