Bordes

Los bordes en CSS tienen 3 caracteristicas basicas: el ancho, el estilo, el color.
El unico que vale la pena explicar es el estilo, y solo hace falta saber que estan solid, una linea continua alrededor del elemento contenedor, dashed, este estilo crea una linea intermitente alrededor,dotted

borde solido
tamaño 1em
color burlywood
borde dashed
tamaño 1.5em
color black
borde dotted
tamaño 0.5em
color dodgerblue

Aparte de estas 3 caracteristicas, tambien disponemos de otras caracteristicas que podremos encontrar bastante utiles.
Para adelantar un poco, en el CSS al editar el "border" estamos editando todas las caras del mismo, sin embargo, tambien es posible editarlo como "border-[top/right/bottom/left]" para editar una cara especifica.

Border-width

Nos permite editar diversamente el ancho cada lado del borde. Podemos de 2 a 4 parametros.

  • 4 parametros: Le asignamos un ancho especifico a cada lado del borde en sentido horario (top>right>bottom>left)

Border radius

Esta caracteristica nos permite redondear las esquinas del borde con cierto porcentaje. Notese lo que sucede cuando el borde es 100%.

10%
25%
100%
En realidad, cuando el contenedor tiene lados de igual tamaño, el efecto del circulo se creará a partir del 50% pero si el contenedor no es equilatero se creará un ovalo, utilizar 100% suele ser mejor practica para crear circulos
