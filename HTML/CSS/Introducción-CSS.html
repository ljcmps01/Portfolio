Introducci칩n a CSS

El CSS es otro lenguaje de interpretacion que va siempre en conjunto al HTML, este nos permite dar estilo color y animaciones a este ultimo.

Metaforicamente el HTML es el contorno del dibujo y el CSS es la pintura

Para hacer esto hay tres metodos:

  1. in line: este metodo consiste en aplicar la propiedad style en la etiqueta correspondiente de la siguiente forma
    < [html tag] style="" >< /[HTML tag]
    Un ejemplo de un div con la propiedad de borde aplicada seria:
    hola

    Este metodo es poco recomendado ya que solo sirve para enriquecer dicha etiqueta en especifico, es decir, estas propiedades no seran transmitidas a otras etiquetas de mismas caracteristicas. Por lo que no es recomendable para propiedades en comun de varios elementos

  2. En el elemento header: Podemos aplicar propiedades generalizadas para un tipo especifico de etiqueta dentro del header, por lo que si aplicaramos el borde al div dentro del header, todos los div dentro de la pagina tendr치 dichas propiedades
    Un ejemplo del header con CSS es:
    <head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS</title>
    </head>
    <style> [HTML TAG]{ properties="";} </style

    Este metodo es util para paginas unicas, sin dudas mejor en general que el metodo anterior pero cuando realizamos un proyecto medianamente complejo con navegacion entre paginas, usualmente queremos que cada pagina tenga un estilo similar, por lo que suele preferirse el siguiente metodo

  3. Archivo externo y referencia en header: Este ultimo metodo suele ser el predilecto a la hora de enriquecer esteticamente una web, consiste en crear un archivo ".CSS" y hacer que el HTML se refiera a el desde el header con el tag:
    <link rel="stylesheet" href="[ubicacion de archivo]">

    Este metodo es el predilecto ya que creando un unico archivo podemos usarlo como "plantilla" de estilo para todas las paginas del proyecto.

    Para enriquecer los tags dentro del archivo CSS se debe editar cada etiqueta como en el metodo anterior

Normalize.CSS

Cada navegador procesa la informacion dentro de un CSS de diferentes formas, normalize es una libreria de CSS que se encarga justamente de igualar el estilo CSS alrededor de todos los navegadores

Selectores

Los selectores son la forma de indicarle a nuestro archivo .CSS a que elementos del HTML queremos cambiarle el estilo, por ejemplo para "decorar" dos parrafos con diferentes caracteristicas

Para lograr esto hay 3 formas:
  • Por tipo de elementos: nos permite darle formato equitativo a todos el texto perteneciente a un mismo tipo de elementos, es decir, este formato sera compartido por todos los elementos iguales
  • Por clases: a침adiendo la propiedad class="[nombre]" a la etiqueta de apertura del tipo de elemento podemos hacer que todos los elementos pertenecientes a esta misma clase tengan cierto formato en comun,
    Decimos cierto formato ya que dentro de una misma clase podemos definir diferentes formatos para diferentes elementos, por ejemplo dentro de la clase "estilizado" podemos hacer que todos los elementos tengan fondo rosa y fuente marron, pero ademas poner una clausula que ademas de este formato en comun, a los encabezados se les aplique cursiva.
    Para referenciar a una clase en el css debemos a침ardir el prefijo . antes del nombre de la clase
  • Por id: Sirve para elementos singulares mas que nada, por ejemplo si queremos que una div en especifico tenga un alineamiento y fuente diferente al resto del cuerpo de la pagina.
    Para referenciarlo en el CSS debemos utilizar el prefijo #.
